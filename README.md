# ProyectoFinal-DMC-PhytonForAnalytics
Proyecto: Análisis Exploratorio de Datos (EDA) - Telco Customer Churn

Descripción del Proyecto : El objetivo del proyecto es desarrollar una aplicación usando Streamlit y Github para poder mostrar como analizar, limpiar, transformar y visualizar los datos históricos para identificar patrones asociados a la fuga de clientes, utilizando un enfoque exploratorio y visual. Para esto se deverá de subir un archivo .csv con toda la información.

Instrucciones de ejecución :
Paso 1: Preparar el repositorio en GitHub
Asegúrate de que tu repositorio público en GitHub contenga estrictamente los siguientes archivos en la carpeta raíz:   app.py (El código fuente principal de la aplicación).  
requirements.txt (Con las librerías necesarias: streamlit, pandas, numpy, matplotlib, seaborn).  
TelcoCustomerChurn.csv (El dataset necesario para el análisis).  

<img width="954" height="398" alt="image" src="https://github.com/user-attachments/assets/2f480e55-37b1-4d1d-bd89-1e7d38987872" />

Subir los archivos de portal.png y LogoPython.png que son necesarios para visualizar la personalización del portal.

Paso 2: Iniciar sesión en Streamlit Cloud
Dirígete a share.streamlit.io. Haz clic en "Sign in" (o "Get started" si es tu primera vez) y conecta tu cuenta de GitHub. Esto le dará permiso a Streamlit para leer tus repositorios.

Paso 3: Crear una nueva aplicación
Una vez dentro de tu panel de control de Streamlit, haz clic en el botón "New app" (Nueva aplicación). Si te lo solicita, autoriza la integración con GitHub.

Paso 4: Configurar el despliegue
Llena el formulario con la siguiente información:
Repository: Selecciona el repositorio donde subiste este proyecto : ProyectoFinal-DMC-PhytonForAnalytics
Branch: Selecciona la rama principal (usualmente main o master).

Main file path: Escribe app.py (o la ruta donde esté tu archivo principal).

Paso 5: ¡Desplegar!
Haz clic en el botón "Deploy!".
Streamlit Cloud comenzará a provisionar un servidor, leerá tu archivo requirements.txt para instalar Pandas, Seaborn y las demás librerías, y finalmente ejecutará tu código. Este proceso puede tardar un par de minutos.

Una vez finalizado el proceso, tu aplicación estará viva en internet. Podrás acceder a ella desde cualquier dispositivo mediante el enlace web generado y compartirlo en tu documento final.
