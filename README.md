# ProyectoFinal-DMC-PhytonForAnalytics
Proyecto: Análisis Exploratorio de Datos (EDA) - Telco Customer Churn

Descripción del Proyecto : El objetivo del proyecto es desarrollar una aplicación usando Streamlit y Github para poder mostrar como analizar, limpiar, transformar y visualizar los datos históricos para identificar patrones asociados a la fuga de clientes, utilizando un enfoque exploratorio y visual. Para esto se deverá de subir un archivo .csv con toda la información.
